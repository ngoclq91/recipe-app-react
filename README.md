## 1. Yêu cầu:
- Sử dụng webpack để create & build app
- Sử dụng bower để install các lib thứ 3 vào project
- Kỹ thuật cache các dependencies libs trong webpack

### 2. Run:
```bash
    > $ npm install
    > $ bower install
    > $ npm start   // run with dev env
    > $ npm build   // run to build
```
